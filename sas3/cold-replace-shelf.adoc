= Substitua a frio uma prateleira DS212C, DS224C ou DS460C
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
É necessário realizar uma substituição a frio da prateleira ao substituir uma prateleira de unidades em um sistema de produção que tenha discos em uso. Este é um procedimento disruptivo que exige que você interrompa os controladores em seu par de HA.

Este procedimento aplica-se a prateleiras com módulos IOM12, IOM12B ou IOM12C.

Use o artigo da base de dados de Conhecimento da NetApp https://kb.netapp.com/onprem/ontap/hardware/How_to_replace_a_shelf_chassis_using_a_cold_shelf_removal_procedure["Como substituir um chassi de prateleira usando um procedimento de remoção de prateleira fria"] .
