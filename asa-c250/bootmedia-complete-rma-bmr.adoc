= Devolva a mídia de inicialização com falha para NetApp - ASA C250
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Se um componente do seu sistema ASA C250 falhar, devolva a peça defeituosa para NetApp. Consulte a página  https://mysupport.netapp.com/site/info/rma["Devolução de peças e substituições"] para mais informações.
