= Devolva a mídia de inicialização com falha para NetApp - AFF A250
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Se um componente do seu sistema de storage AFF A250 falhar, devolva a peça defeituosa para NetApp. Consulte a  https://mysupport.netapp.com/site/info/rma["Devolução de peças e substituições"] página para obter mais informações.
