= Devolva a peça com falha ao NetApp - AFF A220 e FAS2700
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Devolva a peça com falha ao NetApp, conforme descrito nas instruções de RMA fornecidas com o kit. Consulte a https://mysupport.netapp.com/site/info/rma["Devolução de peças e substituições"] página para obter mais informações.

O sistema FAS2700 suporta apenas procedimentos manuais de recuperação de mídia de inicialização. A recuperação automática de mídia de inicialização não é suportada.
