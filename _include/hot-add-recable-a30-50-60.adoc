Ao adicionar uma segunda prateleira a um par de HA A30, C30, A50 ou C60, é necessário refazer a fiação da primeira prateleira entre os módulos de E/S compatíveis com RoCE nos slots 3 e 1 de cada controlador.

O procedimento pressupõe que a prateleira existente esteja cabeada a um módulo de E/S compatível com RoCE no slot 3 de cada controlador.

.Passos
. Recable o primeiro shelf nos módulos de I/O compatíveis com RoCE nos slots 3 e 1 de cada controlador.
+
.. No controlador A, mova o cabo do slot 3 porta b (e3b) para o slot 1 porta b (e1b).
.. Repita o mesmo movimento do cabo no controlador B.


. Verifique se a gaveta cabeada está cabeada corretamente usando https://mysupport.netapp.com/site/tools/tool-eula/activeiq-configadvisor["Active IQ Config Advisor"^]o .
+
Se forem gerados erros de cabeamento, siga as ações corretivas fornecidas.
