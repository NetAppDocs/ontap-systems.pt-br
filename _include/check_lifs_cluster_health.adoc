Antes de retornar o nó de substituição ao serviço, verifique se as interfaces lógicas estão em suas portas iniciais, verifique a integridade do cluster e redefina a giveback automática.

.Passos
. Verifique se as interfaces lógicas estão reportando ao servidor doméstico e às portas:
+
`network interface show -is-home false`

+
Se alguma interface lógica estiver listada como falsa, reverta-a para suas portas originais:

+
`network interface revert -vserver * -lif *`

. Verifique a integridade do seu cluster.  Veja o https://kb.netapp.com/on-prem/ontap/Ontap_OS/OS-KBs/How_to_perform_a_cluster_health_check_with_a_script_in_ONTAP["Como realizar uma verificação de integridade do cluster com um script no ONTAP"^] Artigo da Base de Conhecimento.
. Se a giveback automática foi desativada, reative-a:
+
`storage failover modify -node local -auto-giveback true`
