Ao adicionar uma segunda prateleira a um par de HA A700, é necessário refazer a fiação da primeira prateleira, conectando os cabos nas duas portas compatíveis com RoCE nos slots 3 e 7 de cada controlador.

O procedimento pressupõe que a prateleira existente esteja cabeada aos módulos de E/S compatíveis com RoCE no slot 3 de cada controlador.

.Passos
. Recable o primeiro shelf nos dois conjuntos de portas compatíveis com RoCE no slot 3 e no slot 7 de cada controlador.
+
.. No controlador A, mova o cabo do slot 3 porta b (e3b) para o slot 7 porta b (E7B).
.. Repita o mesmo movimento do cabo no controlador B.


. Verifique se a gaveta cabeada está cabeada corretamente usando https://mysupport.netapp.com/site/tools/tool-eula/activeiq-configadvisor["Active IQ Config Advisor"^]o .
+
Se forem gerados erros de cabeamento, siga as ações corretivas fornecidas.
