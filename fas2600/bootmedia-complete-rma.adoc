= Devolver a peça com falha ao NetApp - FAS2600
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Devolva a peça defeituosa à NetApp, conforme descrito nas instruções de RMA fornecidas com o kit. Veja o https://mysupport.netapp.com/site/info/rma["Devolução de peças e substituições"] Para mais informações, consulte a página. O sistema FAS2600 suporta apenas procedimentos manuais de recuperação de mídia de inicialização. A recuperação automática de mídia de inicialização não é suportada.
