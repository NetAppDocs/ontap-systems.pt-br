= Retornar a mídia de inicialização com falha para a NetApp - ASA C800
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Se um componente no seu sistema ASAF C800 falhar, devolva a peça com falha para a NetApp. Consulte a  https://mysupport.netapp.com/site/info/rma["Devolução de peças e substituições"] página para mais informações.
